## ✨ Design Features

- **Full-screen sections** with distinct visual identities (no single endless scroll)
- **Bento grid** skill cards with hover lift effects
- **Interactive project filter** (All / Flutter / Android / AI-Web)
- **Typed text animation** cycling through specializations
- **IntersectionObserver** scroll-triggered animations
- **Frosted glass navbar** on scroll
- **Cursor glow effect** tracking mouse position
- **Scroll progress bar** at top of page
- **Dark premium palette** — electric violet (#8b5cf6) + cyan (#22d3ee)
- **Mobile responsive** down to 320px

## 📦 Running Locally
